Skeleton Extras

Skeleton Extras is a set of UI components built on Skeleton. It aims to provide simple to use responsive components that are not bloated with unnecessary features. Please note that there is (currently) no guarantee these components will work across multiple operating systems or browsers.

View the source code of this file for annotated usage of the components.
